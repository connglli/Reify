# GraphFuzz

## Preriquests

Z3

## Build

```
# Build all targets
make all

# Build function generation target
make fgen

# Build inter-procedural program generation target
make interpgen
```

## Generate

```
# Generate a set of functions
make genfuncset

# Generate a set of programs supporting inter-procedural functions
make geninterpset
```
